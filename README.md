# EmambaIR



### Installation
This implementation based on [BasicSR](https://github.com/xinntao/BasicSR) which is a open source toolbox for event-guided image reconstruction tasks. 

```python
python 3.10.8
pytorch 2.1.2
cuda 12.1
causal-conv1d 1.0.0
mamba_ssm 1.0.1
```

```
conda install packaging
pip install causal-conv1d==1.0.0
pip install mamba_ssm==1.0.1
pip install -r requirements.txt
python EmambaIR.py
```
